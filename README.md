# CRM Analytics Dashboard

Full-stack CRM analytics starter with a dark dashboard UI, role-based login, PDF report export, email notification support, predictive sales analysis, and churn prediction.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- JWT cookie authentication
- PDF report generation with `pdfkit`
- Email delivery via `nodemailer`

## Run locally

1. `npm install`
2. Copy `.env.example` to `.env.local` and set `CRM_AUTH_SECRET`
3. `npm run dev`
